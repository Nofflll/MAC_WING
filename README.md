# MACWING - Визуализатор САХ (Средней Аэродинамической Хорды)

Профессиональный инструмент для авиационных инженеров и авиамоделистов, предназначенный для расчета и визуализации Средней Аэродинамической Хорды (САХ) крыльев сложной формы.

## Основные возможности

*   **Многосегментные крылья**: Поддержка расчета крыльев, состоящих из нескольких трапециевидных сегментов.
*   **Схема «Утка» (Canard)**: Специальный модуль для расчета эквивалентной хорды и фокуса для самолетов схемы «утка».
*   **Интерактивная визуализация**: Изменение параметров в реальном времени с мгновенным обновлением графика (используется Matplotlib).
*   **Точные вычисления**: Использование библиотеки `decimal` для минимизации погрешностей округления.
*   **Трансформации**: Возможность поворота всей модели вокруг произвольной точки (pivot) для анализа геометрии под углами.
*   **Геометрический метод**: Визуализация классического метода диагоналей для нахождения САХ.

## Установка

Для запуска проекта необходим Python 3.8+ и установленные зависимости.

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-username/macwing.git
    cd macwing
    ```

2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

## Запуск

Запустите основной файл приложения:
```bash
python MACWING.py
```

## Использование

1.  **Главное окно**: Позволяет добавлять несколько крыльев и сегментов к ним. Вводите параметры (хорды, размах, стреловидность) в таблицу.
2.  **Управление**: Используйте поля "Angle" и "Pivot" для поворота модели.
3.  **Окно «Утка»**: Нажмите соответствующую кнопку (если она активна в коде) или используйте специализированный модуль для расчета взаимодействия переднего и заднего крыла.

## Математическая база

Программа реализует:
- Нахождение координат САХ через пересечение диагоналей модифицированного контура.
- Рекурсивное объединение сегментов крыла для получения общей САХ.
- Расчет эквивалентной хорды с учетом разности углов установки (Δφ) и вертикального смещения (N).
